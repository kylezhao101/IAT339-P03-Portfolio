GitHub View the site

This project required the creation of a fictional keyboard brand and its website. Pages for product listings, landing page, about us, and checkout process have been created. This project began with the creation of a style guide, followed by the website being built using these elements.

As the website needed to be responsive for varying window sizes and viewing devices, by using media queries, elements such as this footer are re-arranged to fit the new dimensions.

As this is one of my first coding projects, I learned how to Google effectively in order to learn and debug new and unfamilliar APIs.

Branding

Finally, the KeyLuxe brand identity is outlined in the style guide of the website. With the selection of specific colours for difference interaction methods and implications, as well as typography and modern and elegant usage of photography, a strong brand identity for KeyLuxe was created reflecting the brands core values.

My Learning Outcomes

  • Designing for accessibility

    • During the development of, I learning to consistently test the website with various browser dimensions. I did this to ensure the website's layout would appear organized throughout varying devices. Throughout this testing I would find certain points at which the layout of a section would break or ruin the overall appearance, at which I would use a media query to adjust it.


  • File Organization

    • As the website had varying pages, elements, and specific layouts, the overall amount of class names and line number of the stylesheet greatly increased. In order to keep a clean and organized stylesheet, I implemented a naming system for classes and grouped them so I could easily find them. Finding my desired css selector was further sped up through utilizing the (ctrl + f) search box for keywords.
